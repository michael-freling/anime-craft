# Product Spec: [Feature Name]

## Problem Statement

Anime gets attention from many people aruond the globe, but it's hard for beginners to learn how to create anime.
This prototype app is to focus on people to practice and improve what they want to draw in anime, notably drawing and animation.

### Problem

The base of anime is drawing, but it's so hard to learn how to draw.
In this drawing, it'll improve drawing practice by themselves.

## Goals

- Users can learn what they want to draw and what are differences between what they drew and what they want to draw.
- Users can learn how to draw lines
- Users can learn how to add colors on line drawings
- A system has feedback to users that users can easily understand and learn from the feedback
- A system has feature to start short sessions repeatedly so users can practice many drawings quickly

## Non-Goals

- Educational contents. There are already a lot of contents out there that professional animators teach how to draw.
